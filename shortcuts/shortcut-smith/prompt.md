# Shortcut Smith — prompt

> A plain-English recipe of this shortcut, auto-generated from its actions so it
> always matches what the shortcut really does. Read it before you install — or hand
> it to your own agent to rebuild the shortcut from scratch.

An agentic shortcut that builds shortcuts for you: describe what you want, it asks a model for a step-by-step recipe (in this repo's prompt format) and saves it to build or submit. Shortcuts making shortcuts.

## Steps

1. Asks you for input (device-local).
2. Sends a network request / downloads from a URL — possible exfiltration.
3. Shows a result on screen.
4. Appends text to one of your own Notes (device-local, no network).
5. Posts a local notification.

**Install:** https://www.icloud.com/shortcuts/REPLACE_WITH_REAL_ID
